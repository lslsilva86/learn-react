# What React Can Do (React 19)

## 0.0 JavaScript ES6+ Prerequisites

### 0.1 `let` & `const`
Block-scoped variable declarations replacing `var`; `const` for immutable bindings, `let` for reassignable ones.

### 0.2 Arrow Functions
Concise function syntax that lexically binds `this`, commonly used in React callbacks and hooks.

### 0.3 Destructuring (Array & Object)
Extracting values from arrays or objects into variables; heavily used with props and state.

### 0.4 Spread & Rest Syntax
Spreading iterables into new arrays/objects or collecting remaining values; essential for immutable state updates.

### 0.5 Template Literals
String interpolation using backticks for cleaner dynamic string construction.

### 0.6 ES Modules (`import` / `export`)
Standard JavaScript module system for splitting code across files; the foundation of every React project.

### 0.7 Array Methods (`map`, `filter`, `reduce`, `find`)
Functional methods for transforming and querying data; `map` in particular is used constantly for rendering lists in React.

### 0.8 Promises & Async/Await
Handling asynchronous operations cleanly; critical for data fetching inside React components.

### 0.9 Optional Chaining (`?.`)
Safely accessing deeply nested object properties without throwing errors when a value is `null` or `undefined`.

### 0.10 Nullish Coalescing (`??`)
Returning a fallback value only when the left side is `null` or `undefined`, unlike `||` which triggers on any falsy value.

### 0.11 Short-Circuit Evaluation (`&&`, `||`)
Using logical operators for conditional rendering and default values inside JSX.

### 0.12 Ternary Operator
Inline conditional expression widely used for conditional rendering in JSX.
