# What React Can Do (React 19)

## 13.0 Developer Tooling
React DevTools (browser extension) lets you inspect component trees, profile renders, and debug state.

### 13.1 React DevTools
Browser extension for inspecting component trees, viewing props/state, and debugging renders.

### 13.2 Profiler
Built-in tool in React DevTools for measuring rendering performance and identifying bottlenecks.

### 13.3 Strict Mode
`<React.StrictMode>` highlights potential issues in development by intentionally double-invoking renders and effects.

### 13.4 Hot Module Replacement (HMR)
Updates components in the browser instantly during development without a full page reload.

### 13.5 Fast Refresh
React-specific HMR that preserves component state across edits during development.

### 13.6 ESLint Plugin for React
Linting rules specifically for React to catch common mistakes like missing keys or incorrect hook usage.

### 13.7 TypeScript Support
Full TypeScript integration for type-safe components, props, state, and hooks.

### 13.8 Testing with React Testing Library
A testing utility that encourages testing components from the user's perspective rather than implementation details.

### 13.9 Jest
A JavaScript testing framework commonly used with React for unit and integration tests.

### 13.10 Storybook
A tool for building and documenting UI components in isolation, independent of the app.

### 13.11 Vite & Create React App
Tooling for scaffolding and bundling React projects quickly during development.
