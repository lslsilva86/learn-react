# What React Can Do (React 19)

## 12.0 Cross-Platform
Via React Native, the same component model extends to iOS and Android apps. React Native for Web brings it back to browsers.

### 12.1 React Native
Builds native iOS and Android apps using React's component model with native UI primitives.

### 12.2 React Native for Web
Runs React Native components in the browser, enabling code sharing across mobile and web platforms.

### 12.3 Expo
A framework built on top of React Native that simplifies building and deploying cross-platform apps.

### 12.4 React Native Windows & macOS
Extends React Native to build native desktop apps for Windows and macOS.

### 12.5 Shared Business Logic
Sharing non-UI logic like hooks, state management, and utilities across web and native platforms.

### 12.6 Universal Components
Writing components that render correctly on both web and native with platform-specific adaptations.

### 12.7 React Three Fiber
A React renderer for Three.js, enabling 3D graphics and experiences using React's component model.

### 12.8 React PDF
Renders React components as PDF documents for generating printable or downloadable files.
